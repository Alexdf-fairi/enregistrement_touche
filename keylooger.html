Introduction

Keylogger V2 is a Python-based keylogger tool designed for monitoring keyboard activity. This project uses structured documentation formats like reStructuredText (RST) or Markdown, which are converted into HTML using Sphinx or similar tools.

Features

  • Written in Python
  • Uses pynput for capturing keyboard inputs
  • Records pressed keys and provides analysis
  • Generates visual outputs such as histograms and keyboard heatmaps using matplotlib
  • Switches recording on/off using the Esc key

Usage

The program starts in idle mode. Press Esc to start recording keystrokes. Press Esc again to stop recording. After two presses of Esc, the program terminates and generates statistics. 

# To start the program, use:
python keylogger_v2.py

Example Code

Below is an excerpt from the Keylogger V2 project:

from pynput import keyboard
from collections import Counter
import matplotlib.pyplot as plt

# Global variables
pressed_keys = []
recording = False
esc_count = 0

# Function to start/stop recording
def handle_key_press(key):
    global recording, pressed_keys, esc_count
    if key == keyboard.Key.esc:
        esc_count += 1
        if esc_count == 2:
            return False  # Exit program
        recording = not recording
    elif recording:
        if hasattr(key, 'char') and key.char:
            pressed_keys.append(key.char.lower())

# Run keylogger
with keyboard.Listener(on_press=handle_key_press) as listener:
    listener.join()
